# Geofence_Events (tabla Geofence_Events)
No existen endpoints publicos para esta tabla. Los eventos se crean de forma automatica cuando se procesan ubicaciones y se detecta ENTER o EXIT en las geocercas.

- Insertados por: servicio geofence.service al llamar /api/locations/sync.
- Usados para: historico y generacion de alertas.

Si necesitas exponerlos, crea rutas nuevas (GET/POST) y documentalas aqui.
